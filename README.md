# TechWave Podcast

TechWave Podcast is a responsive podcast landing page designed to showcase podcast episodes, hosts, and community content through a modern dark-themed interface.

## Live Demo
https://mohammedrakibcodes.github.io/B13-A2/

## Features
- Responsive podcast landing page
- Interactive hero section
- Featured podcast episodes
- Embedded YouTube videos
- Host profile section
- Statistics and feature cards
- Smooth hover animations
- Mobile-friendly navigation

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Google Fonts

## Sections
- Hero Section
- About Podcast
- Statistics
- Featured Episodes
- Host Section
- Footer

git clone https://github.com/mohammedrakibcodes/B13-A2.git
