# 🎙️ TechWave Podcast

A modern and responsive podcast landing page built with HTML and CSS.
TechWave is designed to showcase podcast episodes, hosts, platform integrations, and community engagement with a premium dark-themed UI.

## 🌐 Live Demo

🔗 Live Site: https://mohammedrakibcodes.github.io/B13-A2/

---

## ✨ Features

* Fully responsive modern layout
* Dark gradient podcast UI
* Interactive hero section
* Podcast statistics section
* Feature highlight cards
* Embedded YouTube podcast episodes
* Host profile section
* Social media integration
* Smooth hover animations
* Mobile-first responsive design
